### Features

- [x] Enter and save lyrics quotes
- [x] Progress bar shows number of quotes entered

### How to Run
```
$ git clone https://github.com/barclayd/Lyrics-Snippets
$ cd Lyrics-Snippets
$ npm install
$ npm run serve
```
Open [localhost:8080](http://localhost:8080/)
